# Python Functions

This folder contains a complete and structured implementation of
Python function concepts, covering fundamentals to early-intermediate level.

The focus is on understanding how functions work, how arguments and return
values are handled, and how functions help organize reusable logic.

---

## Concepts Covered

### Function Basics
- Defining and calling functions
- User-defined functions
- Built-in functions
- Using the `help()` function

### Function Arguments
- Positional arguments
- Default arguments
- Keyword arguments
- `*args` (variable positional arguments)
- `**kwargs` (variable keyword arguments)

### Return Values
- Returning a single value
- Returning multiple values (tuple unpacking)

### Variable Scope
- Local variables
- Global variables
- Using the `global` keyword

### Special Function Types
- Lambda (anonymous) functions
- Recursive functions
- Nested functions
- Returning functions from functions

---

## Purpose

This section is designed to:
- Build a strong foundation in Python functions
- Demonstrate clean and readable code structure
- Prepare for Object-Oriented Programming and DSA

Each file focuses on one concept and includes simple examples
to clearly explain the logic.
